Bible Desktop...

BibleDesktop is a free Bible study tool that allows you to dig deeply into God's Word. BibleDesktop is available for Windows (98SE to Windows 7), Mac OS X, and more. Its minimum system requirements enable it to run on most computers today. BibleDesktop uses the JSword project to read modules in the Sword format, so there are a huge range of Bibles, commentaries, dictionaries, devotionals, and books — one of the largest collections available. For information on how to download more books and Bible versions, please click on the download button above to begin the download process.

BibleDesktop is open source software, so it is freely available to everyone. It is licensed under the GNU General Public License.

For information on how to download more books and Bible versions, please click on the ADD BOOKS button below to begin the process of adding content.

           

Installation with the internet:

  1. Click the DOWNLOAD button above. This will open a browser to a website where you can download the latest version.
  2. Follow the Prompts to Complete Installation
Installation without the internet:
  1. Navigate to the Prophecy Foundations DVD drive through My Computer (Windows) or Finder (Mac)
  2. Select the "BibleDesktop" folder
  3. Select the folder that matches your operating system (windows or mac) and click on the installation file for BibleDesktop.
  4. Follow the prompts to complete installation.
Documentation:
To view the manual for BibleDesktop click the MANUAL button above.
